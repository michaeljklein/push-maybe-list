# push-maybe-list

Implementation of pushing lists through `Maybe`
with tests.

```haskell
partMaybe :: [Maybe a] -> [[Maybe a]]

dartMaybe :: [[Maybe a]] -> [Maybe [a]]

pushMaybe :: [Maybe a] -> [Maybe [a]]
```


## Tests

Run tests with

```bash
stack test
```
